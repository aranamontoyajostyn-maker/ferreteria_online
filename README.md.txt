# Sistema de Gestión de Ferretería
Sistema web desarrollado para el control de inventario y punto de venta.

## Funcionalidades
- **Gestión de Inventario:** Altas, bajas y modificación de productos.
- **Punto de Venta:** Carrito de compras con validación de stock en tiempo real.
- **Control de Acceso:** Login con roles definidos (Admin / Ventas).
- **Seguridad:** Gestión de sesiones y protección de rutas.

## Tecnologías
- PHP 
- MySQL
- Bootstrap 5 (Diseño responsivo)

## Instalación
1. Clonar el repositorio.
2. Importar el archivo SQL en tu base de datos local.
3. Configurar `conexion.php` con tus credenciales.
